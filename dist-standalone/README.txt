# MatteFlyt - Game Instructions

## How to Play

1. **Double-click on `LAUNCH.bat`** to start the game
2. The game will open automatically in your default web browser
3. Enjoy playing!

## System Requirements

- Any modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- Windows, Mac, or Linux
- No additional software required!

## If the Game Doesn't Open

If the batch file doesn't work:
1. Find and open the `index.html` file directly
2. Right-click on `index.html`
3. Select "Open with" and choose your favorite browser
4. The game will launch in that browser

## Troubleshooting

- **Browser shows blank page**: Refresh the page (press F5)
- **Game runs slowly**: Try a different browser or close other applications
- **Images not loading**: Make sure all files in the `assets` folder are present
